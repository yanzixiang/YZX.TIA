# 目的

2.给 TIA 添加一些我想要的功能。

# Openness 和 OpennessScripter
OpennessScripter 只是一个 Openness 的一种应用而已。Openness 只是给用户提供一种操作 TIA 的方式，提供一种从 TIA 向外导出数据和向 TIA 导入数据的方式。
